# Portfólio Pessoal

Olá! Este é o meu portfólio profissional, desenvolvido para centralizar meus projetos, habilidades e trajetória acadêmica/profissional. O objetivo principal é demonstrar minha capacidade técnica tanto no desenvolvimento de sistemas robustos quanto na criação de interfaces modernas e acessíveis.

## Objetivo

Esse portfólio serve como uma vitrine viva para:
* Exibir meus projetos reais e acadêmicos detalhando a stack utilizada em cada um deles.
* Compartilhar minha experiência atual como graduando em Ciência da Computação.
* Facilitar o contato para oportunidades profissionais e colaborações em projetos open-source.

## Tecnologies Utilizadas

O projeto foi construído utilizando o que há de mais moderno no ecossistema Web:

*   **[Next.js](https://nextjs.org/)**: Framework React com suporte a Server Components para performance otimizada e roteamento intuitivo.
*   **[TypeScript](https://www.typescriptlang.org/)**: Adição de tipagem estática para garantir maior segurança e manutenibilidade do código.
*   **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para um design responsivo, limpo e de rápido desenvolvimento.
*   **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca de animações para criar transições fluidas (como o efeito *fade-up* nos cards de projetos).
*   **[Lucide React](https://lucide.dev/)**: Conjunto de ícones leves e consistentes.
*   **ESLint & Prettier**: Para garantir a padronização e qualidade do código seguindo as melhores práticas do mercado.

## Estrutura do Projeto

O código foi organizado seguindo princípios de modularidade e componentização:

*   `/components`: Componentes reutilizáveis da interface (Cards, Seções, Layout).
*   `/data`: Centralização de informações (como a lista de projetos) para facilitar a manutenção de conteúdo sem mexer na lógica da UI.
*   `/lib`: Configurações de animações e funções utilitárias.
